# Assets du site

## Mockup iPhone (optionnel)

Sur la page **Créer ma carte** (`/creer-ma-carte`), un cadre iPhone type 14/15 (SVG) s’affiche par défaut.

Pour utiliser un **vrai mockup photo** (iPhone en image) à la place :

1. Trouve une image **cadre iPhone** avec **zone écran transparente** (PNG) :
   - Par exemple : cadre noir avec un « trou » transparent à l’emplacement de l’écran.
   - Sites utiles : Freepik, Vecteezy, StickPNG (recherche « iPhone 15 frame mockup PNG transparent »).
2. Dépose le fichier ici avec **exactement** ce nom :
   - **`iphone-frame.png`**
3. Redéploie ou rafraîchis la page.

L’image doit faire à peu près **280×577 px** (ou même proportion) pour que l’écran s’aligne. La zone transparente de l’image sera remplacée par l’aperçu des cartes (Classique, Moderne, Élégant).

Si le fichier n’existe pas, le mockup SVG reste affiché.
