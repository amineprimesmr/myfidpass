# Landing AI Web Design Agency (démo)

- **Dev :** `npm run dev --prefix frontend` puis ouvrir  
  **http://localhost:5174/landing-ai-agency.html**
- **Build :** sortie `dist/landing-ai-agency.html` (entrée Vite multi-page).

Stack : React 19, TypeScript, Vite, Tailwind CSS v4, **shadcn-style** `Button` (CVA), **motion** (Framer), **lucide-react**, **hls.js**. Le prompt source **copier-coller à l’identique** est dans [`PROMPT.md`](./PROMPT.md).
